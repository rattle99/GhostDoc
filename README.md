# zlhacks

## Instructions

#### 1. Install openjdk
```
sudo apt install openjdk-21-jdk
```

#### 2. Create & activate python virtual environment

```py
python -m venv .venv
source .venv/bin/activate
```

#### 3. Install dependencies

```
pip install -r requirements.txt
```

#### 4. Run
```
python app.py
```
